EXECUTION ENGINE V2

This system converts structured indicator inputs into trading decisions.

────────────────────────────
INPUT FORMAT
────────────────────────────

The agent receives:

- PAVP state (above VAH / below VAL / inside VA)
- MACD state (momentum up/down/flat)
- Trend Speed state (expansion / contraction)
- ZigZag structure state (bull / bear / break)
- CVD state (buy pressure / sell pressure / divergence)
- ATR state (low / normal / high volatility)

────────────────────────────
PROCESS FLOW
────────────────────────────

STEP 1: STRUCTURE FILTER
- Use ZigZag as primary directional bias
- If structure is unclear → NO TRADE

STEP 2: LOCATION FILTER
- Use PAVP (VAH / VAL / POC)
- Reject mid-value area trades unless strong momentum exists

STEP 3: MOMENTUM CONFIRMATION
- MACD + Trend Speed must agree with structure
- If conflict → reduce confidence score

STEP 4: ORDER FLOW VALIDATION
- CVD must confirm direction OR show acceptable absorption
- Divergence → reduce score heavily

STEP 5: VOLATILITY FILTER
- ATR low → reject breakout trades
- ATR high → require structure confirmation

────────────────────────────
SCORING SYSTEM
────────────────────────────

Start score = 5.0

Adjust:

+ Structure alignment = +2.0
+ Momentum alignment = +1.5
+ Order flow confirmation = +1.5
+ Clean location (VA edges / POC) = +1.0

- Structure conflict = -2.0
- CVD divergence = -2.0
- Chop zone = -1.5
- Low volatility trap = -1.0

────────────────────────────
OUTPUT
────────────────────────────

Return:

- Direction: LONG / SHORT / NO TRADE
- Confidence: 0–10
- Setup Grade: A / B / C / NONE
- Reasoning (structured per indicator)
- Risk State (LOW / MEDIUM / HIGH)

EXECUTION_ENGINE_V2
═══════════════════════════════════════════════════════════

Central decision fusion system.

Combines inputs from all six indicators after enrichment
by the Python signal layer, then produces a final trade
decision with full score breakdown and reasoning.

═══════════════════════════════════════════════════════════
INPUTS EXPECTED
═══════════════════════════════════════════════════════════

Fully populated signal_format.json containing:

  pavp         → poc, vah, val, poc_distance_pct,
                 value_area_position, value_area_width_pct
  zigzag       → structure, bos_signal, direction,
                 last_swing_high, last_swing_low, leg_bar_count
  macd         → macd_line, signal_line, histogram,
                 histogram_state, zero_line_position, signal_cross
  trend_speed  → direction, regime, speed_normalized,
                 price_vs_ema, wave_analysis.*
  cvd          → cvd_direction, divergence.*, aggression.*,
                 cost.*, absorption.type, delta_implied.*,
                 ma.ma_bias
  atr          → volatility_state, percentile_rank,
                 compression, expansion, atr_pct_of_price

═══════════════════════════════════════════════════════════
STEP 1 — REGIME DETECTION (runs before scoring)
═══════════════════════════════════════════════════════════

Classify active regime using combined indicator state:

TRENDING_UP:
  zigzag.structure = BULLISH
  pavp.value_area_position = ABOVE_VA
  trend_speed.direction = BULLISH + regime = EXPANSION
  macd.histogram_state = BULLISH_ACCELERATING
  cvd.cvd_direction = BUYING

TRENDING_DOWN:
  zigzag.structure = BEARISH
  pavp.value_area_position = BELOW_VA
  trend_speed.direction = BEARISH + regime = EXPANSION
  macd.histogram_state = BEARISH_ACCELERATING
  cvd.cvd_direction = SELLING

RANGING:
  pavp.value_area_position = INSIDE_VA
  zigzag.structure = NEUTRAL or alternating
  trend_speed.regime = CONSOLIDATION
  macd crossing frequently

BREAKOUT:
  atr.compression = true → now expanding
  pavp breaking VAH or VAL
  trend_speed.current_ratio_avg > 1.2
  cvd confirms directional flow

REVERSAL / EXHAUSTION:
  trend_speed.regime = EXHAUSTION
  cvd divergence (medium or large) opposing price
  macd.histogram_state = BULLISH_DECELERATING or BEARISH_RECOVERING
  price far from poc (large poc_distance_pct)

If regime is UNCERTAIN (< 2 indicators agree):
  → Apply -2.0 confidence penalty
  → Bias toward NO TRADE

═══════════════════════════════════════════════════════════
STEP 2 — STRUCTURE FILTER (highest priority layer)
═══════════════════════════════════════════════════════════

Use: zigzag.structure, zigzag.bos_signal, pavp.*

Rules:
  BULLISH structure + BOS_UP + ABOVE_VA  → strong long bias  (+25)
  BEARISH structure + BOS_DOWN + BELOW_VA → strong short bias (-25, short)
  Structure NEUTRAL                       → NO TRADE bias
  ZigZag and PAVP conflict                → NO TRADE or heavy penalty (-15)
  bos_signal = NONE + structure unclear   → reduce confidence

═══════════════════════════════════════════════════════════
STEP 3 — MOMENTUM CONFIRMATION
═══════════════════════════════════════════════════════════

Use: trend_speed.*, macd.*

Trend Speed rules:
  regime EXPANSION + direction aligned    → +10
  current_ratio_avg > 1.2                 → +5 bonus (accelerating)
  current_ratio_avg < 0.4                 → -5 (exhaustion)
  regime EXHAUSTION                       → -5 regardless of direction
  price_vs_ema = CROSSING                 → -3 (transition uncertainty)
  dominance opposing structure            → -3

MACD rules:
  BULLISH_ACCELERATING + ABOVE zero       → +5
  BULLISH_ACCELERATING + BELOW zero       → +3 (building)
  BULLISH_DECELERATING                    → +1 (weakening)
  BEARISH_RECOVERING                      → -1
  BEARISH_ACCELERATING + BELOW zero       → -5
  signal_cross in direction               → +3 bonus
  signal_cross opposing                   → -3

═══════════════════════════════════════════════════════════
STEP 4 — ORDER FLOW VALIDATION
═══════════════════════════════════════════════════════════

Use: cvd.*

CVD direction:
  BUYING confirms long bias               → +10
  SELLING confirms short bias             → +10
  NEUTRAL                                 → 0
  opposing direction                      → treat as divergence

Divergence (scale-weighted):
  divergence.large  opposing              → -15
  divergence.medium opposing              → -10
  divergence.small  opposing              → -5

Absorption:
  BUY_ABSORPTION  (buyers win volume, price drops) → +5 long confirm
  SELL_ABSORPTION (sellers win, price rises)       → +5 short confirm

Cost state:
  VERY_HIGH + aligned direction           → +5
  HIGH + aligned                          → +2
  VERY_LOW                                → -5 (weak move)

Delta implied:
  move_ratio > 1.5                        → -5 (overextended)
  move_ratio < 0.5                        → -3 (suppressed, uncertain)

MA bias:
  cvd.ma.ma_bias = ABOVE (long)           → +2
  cvd.ma.ma_bias = BELOW (short)          → +2
  cvd.ma.ma_bias = CROSSING               → 0

═══════════════════════════════════════════════════════════
STEP 5 — LOCATION FILTER (PAVP)
═══════════════════════════════════════════════════════════

value_area_position:
  ABOVE_VA → bullish continuation context       → +5 to +10
  BELOW_VA → bearish continuation context       → +5 to +10
  INSIDE_VA → chop zone                         → -10

poc_distance_pct:
  Near 0% (within 0.1%) → POC magnet active     → -3
  > 1.5% away from POC  → clear of magnet zone  → 0

value_area_width_pct:
  Narrow VA (< 30%)     → compression, breakout risk → note only
  Wide VA  (> 70%)      → extended range, exhaustion risk → -2

═══════════════════════════════════════════════════════════
STEP 6 — VOLATILITY FILTER (ATR — direction-neutral)
═══════════════════════════════════════════════════════════

volatility_state:
  HIGH + expansion = true  → valid continuation context, widen stop note
  LOW  + compression = true → breakout trades BLOCKED (-10 if breakout type)
  NORMAL               → no adjustment

percentile_rank:
  > 90  → extreme volatility, reduce position size note
  < 10  → very low volatility, fakeout risk on breakouts

compression = true alone → breakout readiness flag (no score change)
expansion   = true alone → momentum environment confirmed

ATR never adds or removes directional bias.

═══════════════════════════════════════════════════════════
OUTPUT FORMAT
═══════════════════════════════════════════════════════════

Return EXACTLY per output_schema.txt:

1. Direction:       LONG / SHORT / NO TRADE
2. Confidence:      0–100
3. Entry Grade:     A / B / C / NONE
4. Regime:          active regime classification
5. Score Breakdown: structure / location / momentum /
                    order_flow / volatility / total

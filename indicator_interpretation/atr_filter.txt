═══════════════════════════════════════════════════════════
WHAT THIS INDICATOR DOES
═══════════════════════════════════════════════════════════

Computes the Average True Range — a measure of market
volatility based on the range of each bar including gaps:

  True Range = max(
    high - low,
    abs(high - close[1]),
    abs(low  - close[1])
  )

  ATR = RMA(True Range, 14)

RMA = Wilder's Moving Average (smoother than EMA)
  RMA[i] = (RMA[i-1] * (length-1) + TR[i]) / length

Default settings:
  Length    : 14
  Smoothing : RMA

═══════════════════════════════════════════════════════════
ALL SIGNAL FORMAT FIELDS
═══════════════════════════════════════════════════════════

atr_value           : float  — raw ATR value in price units
atr_pct_of_price    : float  — (atr / close) * 100
                               Normalises ATR across instruments
                               and price levels

percentile_rank     : float  — 0 to 100
                               ATR's rank vs last 100 bars
                               > 80 = HIGH volatility
                               < 20 = LOW volatility

volatility_state    : string
  "HIGH"    percentile_rank >= 80
  "LOW"     percentile_rank <= 20
  "NORMAL"  everything between

compression         : bool
  true  = ATR has contracted for 5+ consecutive bars
          → breakout readiness signal (no score change)
  false = no compression detected

expansion           : bool
  true  = ATR has expanded for 3+ consecutive bars
          → momentum environment confirmed
  false = no expansion detected

═══════════════════════════════════════════════════════════
PYTHON COMPUTATION LOGIC
═══════════════════════════════════════════════════════════

def compute_atr(df, period=14):
    high, low, close = df['high'], df['low'], df['close']
    tr = pd.concat([
        high - low,
        (high - close.shift(1)).abs(),
        (low  - close.shift(1)).abs()
    ], axis=1).max(axis=1)
    return tr.ewm(alpha=1/period, adjust=False).mean()

def get_atr_pct(atr, close):
    return round((atr / close) * 100, 4)

def get_percentile_rank(atr_series, lookback=100):
    window = atr_series.rolling(lookback)
    rank = window.apply(
        lambda x: (x < x.iloc[-1]).sum() / len(x) * 100
    )
    return round(rank, 1)

def get_volatility_state(percentile_rank):
    if percentile_rank >= 80:
        return "HIGH"
    elif percentile_rank <= 20:
        return "LOW"
    return "NORMAL"

def detect_compression(atr_series, bars=5):
    recent = atr_series.iloc[-bars:]
    return all(recent.iloc[i] < recent.iloc[i-1]
               for i in range(1, len(recent)))

def detect_expansion(atr_series, bars=3):
    recent = atr_series.iloc[-bars:]
    return all(recent.iloc[i] > recent.iloc[i-1]
               for i in range(1, len(recent)))

═══════════════════════════════════════════════════════════
SCORING CONTRIBUTION
═══════════════════════════════════════════════════════════

Priority : FILTER ONLY (Volatility modifier, max 0 / min -10)

ATR NEVER adds directional points.
ATR ONLY applies penalties to protect against bad entries.

HIGH + expansion = true      → 0   (valid momentum environment)
HIGH + expansion = false     → -3  (volatility spike, avoid late entry)
NORMAL                       → 0
LOW  + compression = true    → 0   (coiling for breakout)
LOW  + compression = false   → -5  (dead market, breakout trades blocked)
percentile_rank > 90         → -5  additional (extreme spike)
percentile_rank < 10         → -5  additional (dead market)

Compression flag notes only:
  compression = true → add note: "ATR compressing — breakout readiness"
  expansion   = true → add note: "ATR expanding — momentum environment"

═══════════════════════════════════════════════════════════
IMPORTANT BEHAVIOURAL RULES
═══════════════════════════════════════════════════════════

ABSOLUTE RULE: ATR is NEVER directional.
  It does not indicate whether to go long or short.
  It only indicates whether the environment is safe to trade.

HIGH volatility rules:
  → Widen stop loss tolerance
  → Allow continuation setups
  → Avoid mean-reversion entries (price can overshoot)
  → Note position sizing should be reduced

LOW volatility rules:
  → Block breakout trades (high fakeout risk)
  → Favour mean-reversion trades near VA edges
  → Note that ranges can persist longer than expected

COMPRESSION rules:
  → Signals the market is coiling
  → Do NOT trade the compression itself
  → Wait for the breakout confirmation
  → CVD direction on the breakout bar is critical

ATR spike (percentile_rank > 90):
  → Market moved violently — entry is likely late
  → Reversal risk is elevated
  → This is an exhaustion signal, not an entry signal

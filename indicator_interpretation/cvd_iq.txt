═══════════════════════════════════════════════════════════
WHAT THIS INDICATOR DOES — 7 LAYERS
═══════════════════════════════════════════════════════════

Layer 1 — Core CVD (Cumulative Volume Delta)
  Aggregates lower-timeframe (default 1-min) volume
  directionally into the current timeframe bars.
  direction() = +1 if close > prev close (buy aggression)
              = -1 if close < prev close (sell aggression)
  currentCVD = running sum of (volume * direction)
  This is TRUE cumulative delta, not an approximation.

Layer 2 — Divergence Detection (3 scales)
  Detects when price makes a new extreme but CVD does not.
  isDivBull  / isDivBear   → small  divergence (length 2)
  isDivBullMed/isDivBearMed→ medium divergence (length 5)
  isDivBullLar/isDivBearLar→ large  divergence (length 10)

  Bearish divergence = price higher high, CVD lower high
    (selling hidden behind rising price = weakness)
  Bullish divergence = price lower low, CVD higher low
    (buying hidden behind falling price = strength)

Layer 3 — Aggression Analysis (Bar / Day / Swing windows)
  buyAggNow   = aggressive buy volume this bar
  sellAggNow  = aggressive sell volume this bar
  netDeltaNow = buyAggNow - sellAggNow
  buyPctNow   = buyAgg / totalAgg * 100
  imbalanceNow = buyAgg / sellAgg ratio

Layer 4 — CVD Cost Per Tick
  flowPerTickNow = delta required per tick of price movement
  costStateText  = "Very High Cost" / "High Cost" /
                   "Normal Cost" / "Low Cost" / "Very Low Cost"
  Very High Cost = price moving on heavy real participation
  Very Low Cost  = price moving with minimal delta (weak/suspicious)

Layer 5 — Absorption Detection
  buyAbsorption  = buyers dominate volume BUT price closes down
    → buyers absorbing sell pressure = bullish hidden strength
  sellAbsorption = sellers dominate volume BUT price closes up
    → sellers absorbing buy pressure = bearish hidden strength

Layer 6 — Delta-Implied Close
  expectedClose = where price should close based on delta
  moveRatio     = actual move / expected move
    1.0 = normal, >1.5 = overextended, <0.5 = suppressed

Layer 7 — CVD Moving Averages
  currentCVDma1 = fast MA of CVD (default EMA 50)
  currentCVDma2 = slow MA of CVD (default EMA 200)
  ma_bias = fast vs slow relationship + CVD position

═══════════════════════════════════════════════════════════
PYTHON COMPUTATION LOGIC
═══════════════════════════════════════════════════════════

def compute_cvd(df_ltf):
    """Compute CVD from lower-timeframe OHLCV data."""
    direction = np.sign(df_ltf['close'] - df_ltf['close'].shift(1))
    delta = df_ltf['volume'] * direction
    return delta.cumsum()

def get_cvd_direction(net_delta, total_agg, threshold=0.55):
    buy_pct = (net_delta + total_agg) / (2 * total_agg)
    if buy_pct > threshold:
        return "BUYING"
    elif buy_pct < (1 - threshold):
        return "SELLING"
    return "NEUTRAL"

def get_cost_state(cost_per_tick, cost_history):
    rank = percentile_rank(cost_history, cost_per_tick)
    if rank >= 90: return "VERY_HIGH"
    if rank >= 75: return "HIGH"
    if rank <= 10: return "VERY_LOW"
    if rank <= 25: return "LOW"
    return "NORMAL"

def detect_absorption(buy_agg, sell_agg, close, open_):
    buy_dominant  = buy_agg > sell_agg
    sell_dominant = sell_agg > buy_agg
    if buy_dominant and close <= open_:
        return "BUY_ABSORPTION"
    if sell_dominant and close >= open_:
        return "SELL_ABSORPTION"
    return "NONE"

def get_divergence(price_pivots, cvd_pivots, direction):
    """Detect divergence between price and CVD pivot structure."""
    if direction == 1:  # price making higher high
        if cvd_pivots[-1] < cvd_pivots[-2]:
            return "BEARISH"
    if direction == -1:  # price making lower low
        if cvd_pivots[-1] > cvd_pivots[-2]:
            return "BULLISH"
    return "NONE"

═══════════════════════════════════════════════════════════
IMPORTANT BEHAVIOURAL NOTES
═══════════════════════════════════════════════════════════

- Large CVD divergence OVERRIDES weak momentum confirmation
- Absorption is a COUNTER-INTUITIVE confirmation signal
  (buyers absorbing = price may drop but buyers are in control)
- VERY_LOW cost = price moved cheaply = suspicious move
  Could be a stop hunt or low-participation drift
- move_ratio > 1.5 = price moved more than delta justified
  = overextended, reversal risk elevated
- CVD is context-dependent: trust level varies by regime
  In trending markets: weight lower (trends can persist despite divergence)
  In ranging markets: weight higher (divergence more predictive)

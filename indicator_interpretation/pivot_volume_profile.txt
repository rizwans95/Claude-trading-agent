═══════════════════════════════════════════════════════════
WHAT THIS INDICATOR DOES
═══════════════════════════════════════════════════════════

Anchors a volume profile between consecutive pivot
highs and lows (detected via ta.pivothigh / ta.pivotlow).

For each pivot-to-pivot window it computes:

  POC  — Point of Control: price level with highest
         traded volume. Acts as a market magnet.

  VAH  — Value Area High: upper boundary of the price
         range containing the top N% of traded volume
         (default 68%).

  VAL  — Value Area Low: lower boundary of the same
         value area.

Also tracks whether price has violated (crossed through)
any historical POC or VA level.

═══════════════════════════════════════════════════════════
RAW OUTPUTS FROM SCRIPT
═══════════════════════════════════════════════════════════

poc  : float — price of Point of Control
vah  : float — price of Value Area High boundary
val  : float — price of Value Area Low boundary

═══════════════════════════════════════════════════════════
FIELDS DERIVED IN PYTHON BACKEND
═══════════════════════════════════════════════════════════

poc_distance_pct    : float
  (close - poc) / poc * 100
  Positive = above POC, Negative = below POC
  Near 0% = POC magnet zone active

value_area_position : string
  "ABOVE_VA"  = close > vah  (bullish breakout context)
  "BELOW_VA"  = close < val  (bearish breakdown context)
  "INSIDE_VA" = val <= close <= vah  (chop zone)

value_area_width_pct : float
  (vah - val) / (profile_high - profile_low) * 100
  < 30% = narrow/compressed value area
  > 70% = wide/extended value area

profile_length_bars : int
  Number of bars in the current profile window

traded_volume       : float
  Total volume traded within the current profile window

═══════════════════════════════════════════════════════════
PYTHON DERIVATION LOGIC
═══════════════════════════════════════════════════════════

def get_va_position(close, vah, val):
    if close > vah:
        return "ABOVE_VA"
    elif close < val:
        return "BELOW_VA"
    return "INSIDE_VA"

def get_poc_distance_pct(close, poc):
    return round((close - poc) / poc * 100, 4)

def get_va_width_pct(vah, val, profile_high, profile_low):
    profile_range = profile_high - profile_low
    if profile_range == 0:
        return 0.0
    return round((vah - val) / profile_range * 100, 2)

═══════════════════════════════════════════════════════════
KEY BEHAVIOURAL RULES
═══════════════════════════════════════════════════════════

POC:
  Acts as a price MAGNET, not a breakout signal.
  Price tends to return to POC in ranging conditions.
  Do NOT use POC cross as a directional trade signal.
  poc_distance_pct near 0 = mean reversion risk active.

VAH:
  Potential resistance in bullish moves.
  Acceptance ABOVE VAH (price stays above) = bullish regime.
  Rejection at VAH = high-probability short location.

VAL:
  Potential support in bearish moves.
  Acceptance BELOW VAL = bearish regime.
  Rejection at VAL = high-probability long location.

INSIDE_VA:
  Lower quality trade zone.
  Automatically reduces confidence in scoring.
  Only trade inside VA with extremely strong confluence.

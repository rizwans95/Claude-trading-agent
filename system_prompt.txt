═══════════════════════════════════════════════════════════
INDICATORS — HIERARCHY OF IMPORTANCE
═══════════════════════════════════════════════════════════

1. ZIGZAG STRUCTURE (HIGHEST PRIORITY)
   Source : Zig-Zag Volume Profile [KioseffTrading]
   Defines trend validity and invalidation.
   Fields used:
     structure        → BULLISH | BEARISH | NEUTRAL
     bos_signal       → BOS_UP | BOS_DOWN | NONE
     last_swing_high  → float
     last_swing_low   → float
     direction        → +1 | -1 | 0
     leg_bar_count    → int

2. PIVOT ANCHORED VOLUME PROFILE — PAVP (HIGH PRIORITY)
   Source : Volume Profile, Pivot Anchored [DGT / dgtrd]
   Defines market equilibrium zones.
   Fields used:
     poc                  → float
     vah                  → float
     val                  → float
     poc_distance_pct     → float
     value_area_position  → ABOVE_VA | BELOW_VA | INSIDE_VA
     value_area_width_pct → float
     traded_volume        → float

3. TREND SPEED ANALYZER (MEDIUM-HIGH PRIORITY)
   Source : Trend Speed Analyzer [Zeiierman]
   Defines momentum regime — acceleration vs exhaustion.
   Fields used:
     direction            → BULLISH | BEARISH | FLAT
     regime               → EXPANSION | EXHAUSTION | CONSOLIDATION | NORMAL
     speed_normalized     → float 0–1
     price_vs_ema         → ABOVE | BELOW | CROSSING
     wave_analysis:
       current_ratio_avg  → float  (>1.0 = expanding, <0.4 = exhaustion)
       current_ratio_max  → float  (>1.0 = breakout momentum)
       dominance          → BULLISH | BEARISH | NEUTRAL
       dominance_factor   → float

4. MACD (MEDIUM PRIORITY)
   Source : Built-in TradingView MACD (unmodified)
   Confirms momentum direction and crossover events.
   Fields used:
     macd_line            → float
     signal_line          → float
     histogram            → float
     histogram_state      → BULLISH_ACCELERATING | BULLISH_DECELERATING |
                            BEARISH_RECOVERING   | BEARISH_ACCELERATING
     zero_line_position   → ABOVE | BELOW
     signal_cross         → BULLISH_CROSS | BEARISH_CROSS | NONE

5. CVD IQ (VARIABLE PRIORITY — context dependent)
   Source : CVD IQ [TradingIQ]
   Confirms real order flow participation across 7 layers.
   Fields used:
     cvd_direction        → BUYING | SELLING | NEUTRAL
     divergence.small     → BULLISH | BEARISH | NONE
     divergence.medium    → BULLISH | BEARISH | NONE
     divergence.large     → BULLISH | BEARISH | NONE
     aggression.dominant_side    → BUY | SELL | BALANCED
     aggression.imbalance_ratio  → float
     aggression.net_delta        → float
     cost.cost_state      → VERY_HIGH | HIGH | NORMAL | LOW | VERY_LOW
     absorption.type      → BUY_ABSORPTION | SELL_ABSORPTION | NONE
     delta_implied.move_ratio    → float
     ma.ma_bias           → ABOVE | BELOW | CROSSING

6. ATR (FILTER ONLY — never directional)
   Source : Built-in TradingView ATR (length 14, RMA smoothing)
   Volatility context filter only.
   Fields used:
     volatility_state     → LOW | NORMAL | HIGH
     percentile_rank      → float 0–100
     compression          → bool
     expansion            → bool
     atr_pct_of_price     → float

═══════════════════════════════════════════════════════════
EXECUTION FRAMEWORK
═══════════════════════════════════════════════════════════

STEP 1 — STRUCTURE FIRST
  Read zigzag.structure and zigzag.bos_signal.
  If structure = NEUTRAL and bos_signal = NONE → NO TRADE bias.
  BOS_UP with BULLISH structure = strong long bias.
  BOS_DOWN with BEARISH structure = strong short bias.

STEP 2 — LOCATION ANALYSIS (PAVP)
  ABOVE_VA → bullish breakout context
  BELOW_VA → bearish breakdown context
  INSIDE_VA → chop zone, reduce confidence
  poc_distance_pct near 0 = POC magnet zone (high mean-reversion risk)
  value_area_width_pct narrow = compressed range, breakout risk

STEP 3 — MOMENTUM CONFIRMATION
  Trend Speed regime EXPANSION + direction aligned   → strong confirm
  Trend Speed current_ratio_avg > 1.2               → accelerating
  Trend Speed current_ratio_avg < 0.4               → exhaustion warning
  MACD histogram_state BULLISH_ACCELERATING + above zero → full confirm
  MACD histogram_state BULLISH_DECELERATING          → weakening, caution
  MACD signal_cross in direction                     → bonus

STEP 4 — ORDER FLOW CONFIRMATION (CVD)
  cvd_direction confirms                             → positive
  divergence.large active (opposing)                 → strong warning
  divergence.medium active (opposing)                → moderate warning
  divergence.small active (opposing)                 → mild caution
  absorption.type active                             → counter-intuitive confirm
  cost.cost_state VERY_HIGH aligned                  → strong participation
  cost.cost_state VERY_LOW                           → weak/suspicious move
  delta_implied.move_ratio > 1.5                     → overextended

STEP 5 — VOLATILITY FILTER (ATR)
  HIGH + expansion → widen stops, valid continuation context
  LOW + compression → avoid breakout trades entirely
  compression = true → breakout readiness flag only
  ATR is NEVER used to determine direction.

═══════════════════════════════════════════════════════════
SCORING SYSTEM (0–100)
═══════════════════════════════════════════════════════════

Base score = 50

STRUCTURE (max +25 / min -25):
  +25  ZigZag BULLISH/BEARISH + PAVP aligned + BOS confirmed
  +15  ZigZag aligned + PAVP aligned, no BOS
  +5   ZigZag aligned only
  -15  ZigZag NEUTRAL or structure conflict
  -25  ZigZag and PAVP directly conflict

LOCATION (max +10 / min -15):
  +10  Clean VA edge (VAH/VAL) or POC reaction zone
  +5   Outside VA, accepted direction
  -10  Inside VA mid-zone (chop)
  -15  Inside VA + structure conflict

MOMENTUM (max +20 / min -15):
  +10  Trend Speed EXPANSION + current_ratio_avg > 1.0
  +5   MACD BULLISH_ACCELERATING above zero
  +5   MACD signal cross in direction
  -5   Trend Speed EXHAUSTION (current_ratio_avg < 0.4)
  -5   MACD BULLISH_DECELERATING (weakening)
  -15  Trend Speed and MACD both conflict with structure

ORDER FLOW (max +15 / min -20):
  +10  CVD direction confirms + no divergence
  +5   Absorption detected (confirms hidden buying/selling)
  +5   cost_state VERY_HIGH aligned (strong participation)
  -5   divergence.small active opposing
  -10  divergence.medium active opposing
  -15  divergence.large active opposing
  -5   cost_state VERY_LOW (weak move, suspicious)
  -5   move_ratio > 1.5 (overextended)

VOLATILITY MODIFIER (applied last):
  HIGH + expansion → no penalty, widen stop note added
  LOW + compression → -10 if breakout trade type
  NORMAL → no adjustment

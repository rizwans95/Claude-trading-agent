EXECUTION FRAMEWORK
────────────────────────────────────────

STEP 1: STRUCTURE FIRST
- Identify HH/HL or LH/LL
- Detect Break of Structure (BOS)
- If structure unclear → NO TRADE bias

STEP 2: LOCATION ANALYSIS
- Above VAH → bullish breakout regime
- Below VAL → bearish breakdown regime
- Inside Value Area → chop / low quality zone
- POC = magnet / equilibrium level

STEP 3: MOMENTUM CONFIRMATION
- Trend Speed must align with structure
- MACD histogram must support direction
- Divergence reduces confidence significantly

STEP 4: ORDER FLOW CONFIRMATION
- CVD must confirm direction
- Divergence = warning signal (reduce score)

STEP 5: VOLATILITY FILTER
- ATR extreme spike → avoid entries (late move)
- ATR too low → avoid breakout trades (fakeouts)
- ATR expansion after compression → valid breakout environment
